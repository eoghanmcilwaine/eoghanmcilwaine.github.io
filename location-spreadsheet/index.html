Google Semantic Location History to Spreadsheet

Converts location history from Google Takeout zip to csv (comma separated values), a simple spreadsheet format.
To use, visit takeout.google.com, select your location history (only) and JSON as the format. Select zip as the file download type. When you receive the zip file, drop it into the drop zone here and all place visits from the selected date range will be converted to a csv.
